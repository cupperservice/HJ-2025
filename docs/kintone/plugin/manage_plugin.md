# プラグインの管理
本ドキュメントでは、kintone におけるプラグインの追加、更新、削除手順について説明します。

## プラグインの追加手順

### 1. プラグインファイルの準備

プラグインは通常、zipファイル（拡張子: `.plugin.zip`）として配布されます。プラグインファイルを入手したら、ローカル環境に保存します。

### 2. kintone管理画面へのアクセス

1. kintoneにログインします
2. 画面右上の歯車アイコンをクリックし、**「kintone システム管理」**を選択します
3. 左側メニューの**「プラグイン」**をクリックします

> **注意**: プラグインの追加には、kintoneシステム管理者の権限が必要です。

### 3. プラグインの追加

1. プラグイン一覧画面で、**「読み込む」**ボタンをクリックします
2. **「ファイルを選択」**をクリックし、準備したプラグインファイル（.plugin.zip）を選択します
3. 内容を確認し、**「読み込む」**ボタンをクリックします
4. 確認ダイアログが表示されたら、**「OK」**をクリックします

プラグインが正常に追加されると、プラグイン一覧に表示されます。

### 4. アプリへのプラグイン適用

プラグインを追加しただけでは、アプリでは使用できません。以下の手順で各アプリに適用します。

1. 対象のアプリを開きます
2. 画面右上の歯車アイコンから**「アプリの設定」**を選択します
3. 左側メニューの**「プラグイン」**をクリックします
4. **「プラグインの追加」**をクリックします
5. 利用可能なプラグイン一覧から、追加したいプラグインの**「追加」**ボタンをクリックします
6. プラグインの設定画面が表示される場合は、必要な設定を行います
7. **「保存」**をクリックします
8. アプリ設定画面に戻り、**「アプリを更新」**をクリックします
9. 確認ダイアログで**「更新」**をクリックします

### 5. 動作確認

アプリの更新が完了したら、プラグインが正しく動作するか確認します。

- アプリの画面を開いて、プラグインによる機能拡張が反映されているか確認
- プラグイン設定で指定した動作が正しく行われるか確認

### プラグインの更新・削除

#### プラグインの更新

1. kintoneシステム管理画面の**「プラグイン」**を開きます
2. 更新したいプラグインの**「更新」**ボタンをクリックします
3. 新しいバージョンのプラグインファイルを選択します
4. 内容を確認し、**「更新」**をクリックします

> **注意**: プラグインを更新すると、そのプラグインを使用しているすべてのアプリに影響します。

#### プラグインの削除

1. kintoneシステム管理画面の**「プラグイン」**を開きます
2. 削除したいプラグインの**「削除」**ボタンをクリックします
3. 確認ダイアログで**「OK」**をクリックします

> **注意**: プラグインを削除すると、そのプラグインを使用しているアプリから機能が削除されます。

### トラブルシューティング

#### プラグインが読み込めない場合

- プラグインファイルが正しい形式（.plugin.zip）であることを確認
- ファイルが破損していないか確認
- プラグインがお使いのkintoneバージョンに対応しているか確認

#### プラグインの機能が動作しない場合

- アプリにプラグインが正しく適用されているか確認
- プラグインの設定内容が正しいか確認
- ブラウザのキャッシュをクリアして再度確認
- ブラウザのコンソールでエラーメッセージを確認
